📝 Modern & Responsive Single-Page CV
Bu proje, HTML, CSS ve JavaScript teknolojilerinin tek bir dosyada birleştirilmesiyle oluşturulmuş, hızlı yüklenen ve modern bir dijital özgeçmiş (CV) sitesidir.

🌟 Öne Çıkan Özellikler
Akıllı İçerik Yönetimi: Eğer kullanıcı sol taraftaki bölümlere bir bilgi girişi yapmadıysa, o alanlar sayfada otomatik olarak gizlenir. Bu sayede boş kalan kısımlar görsel bütünlüğü bozmaz.

Tek Dosya Mimarisi: Tüm kod yapısı (HTML, CSS, JS) tek bir index.html dosyasında toplandığı için taşınabilirliği ve kurulumu oldukça kolaydır.

Tam Uyumluluk: Mobil cihazlardan masaüstü bilgisayarlara kadar tüm ekran boyutlarında kusursuz görünür.

🛠️ Teknik Detaylar
Dil: HTML5, CSS3, JavaScript (Vanilla JS)

Yapı: Single File Application (SFA)

Mantık: JavaScript ile DOM manipülasyonu yapılarak boş veri içeren elementlerin display: none özelliği kontrol edilir.
